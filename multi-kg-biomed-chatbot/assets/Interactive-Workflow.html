Bio-Trapi Agent

Bridging the gap between natural language and structured biomedical knowledge. This interactive report demonstrates how we solve identifier hallucination by grounding LLMs in strict ontology standards before querying complex Knowledge Graphs.

The Challenge: "Identifier Hallucination"

Large Language Models (LLMs) often struggle with specific biomedical identifiers. They might know "Tylenol" is a drug, but they guess its ID in a database (e.g., inventing "CHEM:999" instead of the correct "CHEMBL:112").

This disconnect causes queries to fail when hitting strict databases like HetioNet or ROBOKOP. The Bio-Trapi Agent acts as a precise mediator layer.

Failure Mode (Without Agent)

❌ User: "Find targets for ACE2."
🤖 LLM: MATCH (g:Gene {id: "ACE2"}) ...
📉 DB: Error: Node "ACE2" not found. Expected CURIE (e.g., HGNC:13557).
